### About The Company

Media Revolution SRL is a company dedicated to the development of platforms and software, located in Santo Domingo, National District.

### Responsibilities

My primary role involved converting PSD files into functional HTML/CSS designs for ad campaigns across various betting platforms, which were provided to me by companies from different industries across the globe. As I grew more experienced, I expanded my skill set to include the development of ads with the company's UX/UI approach, aimed at enhancing user engagement with the services offered.

### Tools, Tech & Skills

The tools used at the time were: 
- Sublime Text 2 as the main text editor
- MAMP server to display and share the work inside the local network
- HTML and CSS, along with Bootstrap 3. 
- Other tools included Photoshop to extract assets from design files

Some of the skills I learnt and flexed during this time were:
- Responsive Web Design.
- Javascript for making dynamic pop up ads
- Collaborative Problem Solving
