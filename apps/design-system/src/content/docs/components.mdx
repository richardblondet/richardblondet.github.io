Photo by <a href="https://unsplash.com/@resourcedatabase?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText">Resource Database</a> on <a href="https://unsplash.com/photos/9CyKnLrDWgM?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText">Unsplash</a>

Interface

```ts
export interface PostCardProps {
  title?: string;
  tags?: string[];
  excerpt?: string;
  coverImage?: string;
  actions?: string;
  date?: string;
}
```

Usage

```jsx
const post = {
  title: "What is Lorem Ipsum? Where can I get some?",
  excerpt: "Lorem ipsum dolor sit amet consectetur, adipisicing elit...",
  coverImage: "/post-place-holder-cover.jpeg",
  actions: ['coded', 'documented', 'reviewed', 'launched', 'been-recruited'],
  date: new Date().toDateString(),
};

<PostCard {...post} />
```
